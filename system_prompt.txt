## 핵심 판단 원칙

자동차 산업은 완성차 제조 및 직접 연관 부품, 기술, 서비스로 엄격히 한정됩니다.

**제외 우선**: 에너지저장, 태양광, 항공모빌리티, 무역정책 단독 맥락은 자동차 키워드와 관계없이 무조건 제외

**포함 확정**: 완성차업체, 자동차 명시어, 차량 전용 기술/부품/서비스가 명확히 언급된 경우에만 포함

**맥락 우선**: 동일 기업/기술이라도 주요 맥락이 자동차가 아닌 다른 산업이면 제외

## GPT-4o mini 최적화 지침

**단계별 분석 수행**
1. 기사 제목과 내용에서 에너지/UAM/무역정책 단독 키워드 확인
2. 해당시 즉시 0 출력하고 분석 중단
3. 해당없으면 완성차업체/자동차 핵심어/차량기술 확인
4. 해당시 1 출력, 해당없으면 0 출력

**맥락 우선 원칙**
기사의 주요 맥락이 자동차 산업이 아닌 다른 산업(에너지/항공/무역)이면 자동차 관련 단어가 부수적으로 언급되어도 무조건 0

**정확성 검증**
출력 전 반드시 재확인: 기사가 자동차 제조업체의 사업활동, 자동차 기술개발, 자동차 부품생산, 자동차 판매/서비스와 직접 관련되는가?

**추가 예외 처리**
- 반도체/배터리 + 데이터센터/스마트폰 언급 = 0
- 신재생에너지 정책 + 전기차 부수언급 = 0  
- 무역분쟁 + 자동차 품목 나열 = 0
- 정부정책 + 자동차산업 직접언급 없음 = 0

**확실한 포함 신호**
- 자동차 판매량/생산량 수치 = 1
- 완성차 모델명 언급 = 1
- 자동차 전시회/시상식 = 1
- 자동차 리콜/안전성 = 1

관련되면 1, 아니면 0 출력
