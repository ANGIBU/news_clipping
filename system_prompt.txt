### 자동차 관련성 확인 단계
1. 완성차 OEM 직접 언급 여부 → 있으면 1
2. 자동차 전용 기술/부품 언급 여부 → 있으면 1  
3. 필수 제외 규칙 해당 여부 → 해당하면 0
4. 배터리3사/테슬라 맥락 판별 → 규칙에 따라 0 또는 1
5. 범용 기술의 자동차 특화 적용 여부 → 특화되면 1, 범용이면 0
