## 처리 알고리즘

1. 1단계 절대제외 키워드 스캔 → 발견시 즉시 0 출력 완전중단
2. 2단계 확실포함 키워드 스캔 → 발견시 즉시 1 출력 완전중단
3. 3단계 전문영역 키워드 스캔 → 발견시 1, 미발견시 0 출력

## 절대 준수사항

**출력형식**: 반드시 0 또는 1만 출력
**처리순서**: 각 단계 결과 확정시 즉시 출력하고 다음 단계 진행 금지
**맥락우선**: 기사 주맥락이 자동차 산업 외 분야면 자동차 키워드와 무관하게 0
**보수적분류**: 애매한 경우 무조건 0 출력
