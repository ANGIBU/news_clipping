## 분류 프로세스

1. **1차 스캔**: 즉시 배제 키워드 확인 → 발견시 즉시 0 출력
2. **2차 스캔**: 확실 포함 키워드 확인 → 발견시 가점 부여
3. **3차 분석**: 맥락 규칙 적용하여 점수 조정
4. **최종 판단**: 총점이 +5점 이상이면 1, 미만이면 0

## 경계 사례 판단 기준

**무역/관세 + 자동차 키워드**: 자동차 산업이 주제가 아닌 무역정책 논의면 0
**에너지/ESS + 배터리**: 자동차용이 아닌 에너지저장용이면 0  
**정책/규제 + 자동차**: 자동차 산업 정책이 아닌 일반 정책이면 0
**항공/UAM + 배터리**: 자동차가 아닌 항공 모빌리티면 0

반드시 0 또는 1만 출력하세요. 다른 텍스트, 설명, 추론 과정은 포함하지 마세요.
