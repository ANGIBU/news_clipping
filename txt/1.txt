## 특수 맥락 처리 규칙

**배터리 업체 맥락 구분**
LG에너지솔루션, 삼성SDI, SK온, CATL 언급시:
- 완성차업체명 동시 언급 또는 차량용배터리 명시 = 1
- ESS, 태양광, 주택용, 에너지저장 단독 맥락 = 0

**테슬라 맥락 구분**
- 차량, 판매, 생산, 충전소, 모델명, 자율주행 맥락 = 1
- 에너지, 태양광, 파워월, ESS, 에너지생태계 맥락 = 0

**정책 맥락 구분**
- 자동차산업정책, 전기차보조금, 자동차관세 = 1
- 일반 관세정책, 에너지정책, 이민정책 = 0

## 핵심 판단 원칙

**맥락 우선**: 기사의 주요 맥락이 자동차 산업이 아닌 다른 산업이면 자동차 관련 단어가 부수적으로 언급되어도 무조건 0

**직접 연관성**: 완성차 제조업체의 사업활동, 자동차 기술개발, 자동차 부품생산, 자동차 판매 및 서비스와 직접 관련된 경우에만 1

## 경계선 케이스 처리

**확실한 제외 (0)**
- 무역분쟁 + 전기차 품목 나열
- 에너지정책 + 전기차 부수언급
- 테슬라 + 에너지생태계 언급  
- 배터지업체 + ESS/태양광 단독
- 정부정책 + 자동차산업 직접언급 없음
- 한화LG + 테슬라 + 에너지솔루션
- 관세갈등 + 전기차 품목언급
- 행정명령 + 전기차 정책변경

**확실한 포함 (1)**
- 자동차 판매량/생산량 수치
- 완성차 모델명 언급
- 자동차 전시회/시상식
- 자동차 리콜/안전성
- 자동차제조 공장 설립/확장
- 자동차공장건설, 완성차라인증설
- 자동차시장점유율, 차량브랜드순위

## 실행 순서

1. 제목과 내용에서 에너지/UAM/무역정책/반도체/제조업/정부정책 단독 키워드 확인
2. 해당시 자동차 산업 직접 연관성 없으면 즉시 0 출력
3. 해당없으면 완성차업체/자동차 핵심어/차량기술/자동차부품/자동차생산/자동차서비스 키워드 확인
4. 해당시 1 출력, 해당없으면 0 출력

정확성 검증: 기사가 자동차 제조업체의 사업활동, 자동차 기술개발, 자동차 부품생산, 자동차 판매 및 서비스와 직접 관련되는지 최종 확인

관련되면 1, 아니면 0 출력
