## 판단 원칙

1. **1단계 우선**: 제외 키워드 발견시 즉시 0
2. **2단계 확인**: 포함 키워드 발견시 즉시 1
3. **맥락 우선**: 주맥락이 자동차 외 산업이면 0
4. **보수적**: 불확실하면 0

## 경계선 처리

정책+자동차키워드=0 / 에너지+자동차업체=0 / 기술+자동차응용=0 / 자동차산업정책=1

## GPT-4o mini 최적화

- 키워드 매칭 우선, 복잡한 추론 지양
- 동일 키워드 → 동일 출력 (Temperature 0.4 일관성)
- 확실하지 않으면 0 (변동성 최소화)
- 제목→내용 순서 스캔, 조건 만족시 즉시 출력

## 최종 검증

기사가 다음에 해당하는가?
- 자동차 제조업체 사업활동
- 자동차 기술개발
- 자동차 부품생산  
- 자동차 판매서비스
- 자동차 안전성
- 자동차산업정책

해당=1 / 미해당=0

**출력**: 반드시 0 또는 1만 출력. 다른 텍스트 절대 금지.

**지침**: 키워드 매칭 우선, 확실하지 않으면 0, Temperature 0.4 일관성 보장.
